Unit testing is when the components of the software is tested.
It is normally done by the developers.
It is a type of whitebox testing.

Integration testing is combined execution of two or more classes/packages.
It is done by developers.
It is both white box and black box testing.

System testing is testing of the software in it's final configuration.
Done by developers.
It is whitebox testing.

Acceptance testing is testing that is done after the system is fully developed and tested.
Acceptance testing is done by end-user.
Blackbox Testing
